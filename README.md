# МагӀарул мацӀ — Аварско-русский и Русско-аварский словарь

## О проекте
Полнофункциональный словарь аварского языка с интеграцией в Telegram и веб-приложением (Mini App). Проект обеспечивает быстрый поиск переводов с учетом морфологии, предоставляет ежедневную рассылку "Слово дня" и позволяет использовать словарь в веб-формате.

**Репозиторий проекта:** [https://github.com/sharapik26/Avar_dictionary](https://github.com/sharapik26/Avar_dictionary)

## Состав команды
- **Насрулаев Шарапудин**
- **Мусаев Умахан**
- **Магомедов Абдул**
- **Атаханов Набиюлла**

## Основной функционал
1. **Многоуровневый поиск**: Поддержка двустороннего поиска (аварско-русский, русско-аварский).
2. **Telegram Bot**: Интерактивный бот для перевода слов.
3. **Telegram Mini App**: Веб-приложение для работы со словарем внутри Telegram.
4. **Слово дня**: Ежедневная автоматическая рассылка случайного слова для подписчиков (реализовано через `APScheduler`).
5. **API-сервер**: Бэкенд на базе `FastAPI` с отдачей JSON-ответов и статики.

## Архитектура и надежность
- Реализована отказоустойчивая архитектура: единый монолит на `FastAPI` (`Lifespan`), объединяющий API-сервер и Telegram-бота в одном асинхронном процессе.
- Предусмотрена обработка пустых или невалидных запросов, а также перехват ошибок при рассылках (автоматическое удаление заблокировавших бота пользователей).
- Кодовая база соответствует стандарту PEP-8 и содержит докстринги.

## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/sharapik26/Avar_dictionary.git
cd Avar_dictionary
```

### 2. Зависимости
Проект требует Python версии 3.10 и выше. 
```bash
pip install -r requirements.txt
```

### 3. Конфигурация
Определите необходимые переменные окружения:
- `BOT_TOKEN`: Токен Telegram-бота.
- `WEBAPP_URL`: URL развернутого приложения (для Mini App).

### 4. Запуск
Проект объединяет API и Бота в один процесс:
```bash
uvicorn api.main:app --host 0.0.0.0 --port 8000
```
Сервер будет доступен по адресу `http://localhost:8000`.

## Развертывание
Для развертывания проекта в production-среде используйте следующую команду сборки:
```bash
pip install -r requirements.txt
```
Команда запуска:
```bash
uvicorn api.main:app --host 0.0.0.0 --port $PORT
```
Не забудьте настроить соответствующие переменные окружения (`BOT_TOKEN`, `WEBAPP_URL`) на вашем сервере (например, Render).
